# Web Design Agency Website

This is a complete static website package for a premium web design agency.

## Files

- `index.html` - compact conversion-focused homepage with curated service and work previews
- `services.html` - dedicated services detail page
- `work.html` - dedicated AI solutions and automation capabilities page
- `process.html` - dedicated project process page
- `pricing.html` - dedicated pricing and package page
- `contact.html` - dedicated contact and inquiry page
- `datenschutz.html` - bilingual privacy-policy template with labeled legal placeholders
- `agb.html` - bilingual B2B terms template with labeled legal placeholders
- `styles.css` - original responsive component foundation
- `prestige.css` - high-end editorial art direction, page identities, responsive refinements, and motion system
- `assets/` - optimized, people-free WebP artwork focused on web design and digital products
- `script.js` - accessible navigation, persistent German/English translation, consent management, parallax, repeatable reveals, and contact form validation
- `CONTENT-DATA-CHECKLIST.md` - complete list of brand, AI, service, pricing, contact, SEO, and legal data needed for launch

## Missing Data Labels

All unknown business details are intentionally left as bracketed placeholders, for example:

- `[TAGLINE]`
- `[EMAIL]`
- `[BOOKING_LINK]`
- `[LANGUAGE]`
- `[PRIMARY_COLOR]`
- `[FORM_ENDPOINT_OR_CRM]`

Replace these labels with the final brand, contact, pricing, SEO, and integration data before publishing.

## Launch Notes

The site runs by opening `index.html` directly in a browser. For production, connect the contact form to your chosen form endpoint, CRM, or backend route.

## Design Direction

- Editorial near-black and white foundation with electric blue and coral accents
- Full-bleed homepage hero focused on AI-powered web systems
- Compact homepage that directs visitors to focused detail pages instead of repeating them
- Distinct page identities for Services, AI Solutions, Process, Pricing, and Contact
- People-free web design imagery with responsive cropping and subtle hover depth
- Persistent DE/EN switch across every page
- First-party cookie consent tool with localStorage, optional script categories, and editable preferences
- Branded 0.5-second loading screen and new 42 Craft logo across header and footer areas
- Staggered entrances, subtle parallax, animated proof metrics, and scroll progress
- Reduced-motion support, visible focus states, mobile navigation, and overflow protection
